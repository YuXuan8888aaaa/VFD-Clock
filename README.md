   本项目采用STM32F103CBT6作为系统MCU，使用I2C协议连接DS3231独立RTC实现时钟功能，利用四个
独立按键实现调时、显示切换、亮度调节功能。通过SPI协议连接8-MD-06INKM点阵屏来显示开机画面、
时间、日期，通过蜂鸣器提示按键长短按的操作。PCB预留光敏电阻、LED流水灯、OLED屏、送话器、气
压传感器、EPROM接口。 


   该项目软件设计主要包括主程序、应用层子程序、VFD屏显示子程序、RTC配置子程序、蜂鸣器子程
序、按键操作子程序等。主程序是通过调用子程序等来显示开机问候语、时间、日期及调节屏幕亮度的
功能。

   RTC模块的初始化与数据读写设计，由于硬件特性，在首次对 DS3231进行操作之前，需对其进行初
始化，本项目采用的做法是设定好一个准确的时间后，由于备用电池的存在，在系统下电后时间信息在
其寄存器内也不会丢失，所以以后只需从DS3231中读出数据，程序只需从DS3231的各寄存器中读出年
月、日、时、分、秒等数据，再处理即可。



   VFD显示屏的驱动程序设计，显示部分程序的设计步骤为，先确定VFD于STM32的IO接口连接方式和
通讯协议。然后初始化VFD，即向VFD内的驱动芯片写入一系列的设置程序，对硬件进行配置来启动VFD
的显示。最后进行显示字符的位置规划和调整，输出显示内容。
